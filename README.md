## 📌 Ringkasan Proyek (singkat)

Implementasi dan pengujian yang dilakukan pada proyek ini meliputi:

- Simulasi topologi menggunakan **GNS3**
- Konfigurasi layanan **DHCP** dan mekanisme **DHCP Relay** antar subnet
- Penyusunan dan penerapan **static routing** untuk interkoneksi subnet
- Implementasi **DNS Server berbasis BIND9** untuk resolusi lokal
- Konfigurasi perangkat **MikroTik** (router) dan pengujian konektivitas end-to-end
